# resas-codetest

## 概要
RESAS APIを利用した都道府県別の人口推移グラフを表示するアプリケーションです。

## 構成
- Vue.js
- TypeScript
- Vite/Vitest
- Github Actions/Pages

## 環境構築
1. 
```
npm install
```

2. RESAS APIキーを取得し、.env.localに記載。.env.local.template参照。

## コマンド
- デバッグ
```
npm run dev
```

- テスト実行
```
npm run test
```

- テストのカバレッジ表示
```
npm run coverage
```

- lint
```
npm run lint
```

- フォーマット
```
npm run format
```
